# React-Native-Web Boilerplate

# Summary

A boilerplate for build android, iOS and web apps, using React Native

# Running the project

```
npm install
```

## Android

```
npm run android
```

## iOS

```
npm run ios
```

## web

```
npm run web
```

## test

```
npm run test
```

## load data from source

```
npm run load:data
```

## download logos from source

```
npm run load:logos
```
